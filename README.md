# ShumoffSite

Сайт компании **Shumoff Podolsk** — профессиональная шумоизоляция автомобилей в Подольске и Москве. Улучшенная версия (v2) сайта [shumoffpodolsk.com](https://shumoffpodolsk.com/).

## Структура репозитория

```
shumoff-theme/    Кастомная WordPress-тема (шаблоны, стили, скрипты)
shumoff-core/     Плагин: типы записей (услуги, автомобили, кейсы, отзывы),
                  таксономии (марки авто, типы работ) и поля ACF
docs/             Проектная документация: ТЗ, контент-план, карта сайта,
                  план рефакторинга, статические HTML-прототипы (архив)
```

## Требования

| Компонент | Версия |
|-----------|--------|
| WordPress | 6.x    |
| PHP       | 8.0+   |
| ACF Pro   | 6.x (опционально — тема деградирует без него корректно) |

## Установка

1. Скопировать `shumoff-theme/` в `wp-content/themes/`, активировать тему.
2. Скопировать `shumoff-core/` в `wp-content/plugins/`, активировать плагин
   (при активации автоматически сбрасываются правила ЧПУ).
3. Установить и активировать ACF Pro — поля кейсов, услуг, автомобилей и
   отзывов зарегистрированы кодом в `shumoff-core`.
4. Назначить меню на области «Header Menu» и «Footer Menu».
5. Создать статическую главную страницу (Настройки → Чтение) — используется
   шаблон `front-page.php`.

## Настройки сайта

При активном ACF Pro в админке появляется страница **«Настройки сайта»**:
контакты (телефон, e-mail, адрес, часы, мессенджеры), FAQ главной страницы и
комплекты Simple/Medium/Premium. Без ACF используются дефолты, зашитые в
`shumoff-theme/functions.php` (`shumoff_contact()`, `shumoff_get_faqs()`,
`shumoff_get_packages()`).

## Прайс-лист

Матрица цен «тип кузова × зона × комплект» лежит в
`shumoff-core/data/price-list.php` (сгенерирована из внутреннего справочника)
и отдаётся через `shumoff_get_price_table( $body_type )`. Страница автомобиля
строит таблицы цен по полю «Тип кузова». Переопределение — фильтром
`shumoff_price_list`.

## Типы записей

- **services** (`/services/`) — услуги шумоизоляции;
- **cars** (`/cars/`) — марки/модели автомобилей с ценами;
- **cases** (`/cases/`) — портфолио выполненных работ;
- **reviews** — отзывы (без собственных страниц, выводятся блоками).

Таксономии: `car_brand` (`/brand/`) для авто и кейсов, `case_type`
(`/work-type/`) для услуг и кейсов.

## Разработка

Стили: `shumoff-theme/style.css` — базовые стили и компоненты каркаса;
`shumoff-theme/assets/css/main.css` — стили компонентов внутренних страниц.
Скрипты: `shumoff-theme/assets/js/main.js` (без зависимостей).

Внутренние документы компании (прайсы, акты) в репозиторий не коммитятся —
см. `.gitignore`.
