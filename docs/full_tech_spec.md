# ТЕХНИЧЕСКОЕ ЗАДАНИЕ

## Проект: Shumoff v2

### Платформа: WordPress + Bricks Builder + ACF Pro

---

# 1. Цели проекта

Разработать корпоративный сайт студии шумоизоляции автомобилей с возможностью дальнейшего SEO-масштабирования.

Основные задачи:

* получение заявок;
* продвижение услуг шумоизоляции;
* продвижение по моделям автомобилей;
* демонстрация выполненных работ;
* удобное управление контентом без программиста.

# 3. Структура данных

---

## CPT №1 — Услуги

Slug:

```text
services
```

### Поля ACF

| Поле             | Тип      |
| ---------------- | -------- |
| Цена от          | Number   |
| Срок выполнения  | Text     |
| Краткое описание | Textarea |
| Преимущества     | Repeater |
| Материалы        | Repeater |
| Галерея          | Gallery  |
| FAQ              | Repeater |
| Видео URL        | URL      |

---

## CPT №2 — Автомобили

Slug:

```text
cars
```

### Поля

| Поле         | Тип      |
| ------------ | -------- |
| Марка        | Taxonomy |
| Год модели   | Text     |
| Цена Simple  | Number   |
| Цена Medium  | Number   |
| Цена Premium | Number   |
| Особенности  | WYSIWYG  |
| Фото модели  | Image    |

Slug:

# 18. Критерии приемки

Сайт считается завершенным, если:

✅ Все шаблоны работают.

✅ Все формы отправляют заявки в Telegram и Email.

✅ Работает адаптивная версия.

✅ Настроены CPT и ACF.

✅ Настроен Rank Math.

✅ Настроены карта сайта и robots.txt.

✅ PageSpeed соответствует требованиям.

✅ Настроены резервные копии.

✅ Настроены меры безопасности.

## Дополнение, которое я рекомендую включить сразу

Для SEO в 2026 году добавить еще два CPT:

### `reviews`

Отзывы клиентов

### `faq`

База вопросов и ответов

Это позволит автоматически выводить отзывы и FAQ на страницах услуг, автомобилей и кейсов без дублирования контента и значительно упростит дальнейшее продвижение. Также рекомендую сразу предусмотреть мультирегиональность (Подольск, Москва, Домодедово, Чехов, Видное) через отдельные посадочные страницы, если планируется расширение географии.
