# План работ на ближайшие 2 недели

### Неделя 1

1. Развернуть WordPress.
2. Установить Bricks Builder.
3. Настроить CPT и ACF.
4. Создать структуру URL.
5. Подготовить дизайн главной страницы.

### Неделя 2

1. Сверстать главную.
2. Сделать шаблон услуги.
3. Сделать шаблон автомобиля.
4. Сделать шаблон кейса.
5. Загрузить первые 20–30 кейсов.

После этого сайт уже можно открывать для индексации и начинать SEO-продвижение. Следующим документом я бы подготовил **полное техническое ТЗ для разработчика WordPress (Bricks + ACF + CPT + SEO + безопасность)**, чтобы можно было сразу отдавать в работу.
