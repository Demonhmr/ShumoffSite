# Структура данных WordPress (Shumoff v2)

## 1. Custom Post Types

Создаем 3 основных типа записей.

### CPT №1 — Услуги

```text
services
```

Примеры:

* Полная шумоизоляция
* Шумоизоляция дверей
* Шумоизоляция пола
* Шумоизоляция крыши
* Шумоизоляция арок
* Шумоизоляция багажника
* Детейлинг
* Автозвук
* Авточехлы

---

### CPT №2 — Автомобили

```text
cars
```

Примеры:

* Kia Rio
* Geely Monjaro
* Haval Jolion
* Chery Tiggo 7 Pro

---

### CPT №3 — Кейсы

```text
cases
```

Примеры:

* Шумоизоляция Geely Monjaro 2024
* Шумоизоляция Kia Rio 2021

# 3. Поля ACF для услуг

Группа:

```text
Service Fields
```

Поля:

| Поле                   | Тип      |
| ---------------------- | -------- |
| Цена от                | Number   |
| Срок выполнения        | Text     |
| Краткое описание       | Textarea |
| Преимущества           | Repeater |
| Используемые материалы | Repeater |
| Галерея                | Gallery  |
| FAQ                    | Repeater |
| Видео                  | URL      |

---

# 4. Поля ACF для автомобилей

# 6. Связи между сущностями

Самое важное.

## Автомобиль ↔ Кейсы

Например:

```text
Geely Monjaro
```

автоматически показывает:

```text
Кейс №1
Кейс №2
Кейс №3
```

---

## Услуга ↔ Кейсы

Например:

```text
Шумоизоляция арок
```

автоматически показывает все кейсы по аркам.

---

## Автомобиль ↔ Услуги

На странице:

```text
/cars/geely-monjaro/
```

выводятся:

* Полная шумоизоляция
* Арки
* Двери
* Пол

# 10. Контент-план для запуска MVP

### Услуги

6 страниц

### Автомобили

20 страниц

### Кейсы

30 кейсов

### Статьи

10 статей

### Отзывы

15–20 отзывов

---

После этого этапа следующий документ — **ТЗ на главную страницу (структура блоков + тексты + конверсионные элементы)**. Именно главная будет задавать весь дизайн сайта и станет основой для работы дизайнера и верстальщика.
