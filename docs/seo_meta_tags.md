# SEO-структура и ТЗ на мета-теги

Этот документ определяет иерархию URL-адресов сайта и шаблоны генерации мета-тегов для разделов «Услуги» и «Марки автомобилей». 
Цель: максимальный охват коммерческих запросов вида «шумоизоляция [марка] [модель] цена».

## 1. Структура URL (ЧПУ)

Максимальный уровень вложенности для целевых страниц не должен превышать 3 клика от Главной.

* **Услуги:** `https://shumoffpodolsk.com/services/`
  * Конкретная услуга: `.../services/shumoizolyaciya-dverej/`
* **Марки авто:** `https://shumoffpodolsk.com/cars/`
  * Страница марки: `.../cars/kia/`
  * Страница модели: `.../cars/kia/rio/` (Оптимально) ИЛИ `.../cars/kia-rio/` (Допустимо)
* **Кейсы:** `https://shumoffpodolsk.com/cases/`
  * Конкретный кейс: `.../cases/shumoizolyaciya-geely-monjaro-2023/`
* **Блог:** `https://shumoffpodolsk.com/blog/`

## 2. Шаблоны мета-тегов для страниц марок и моделей авто (CPT: cars)

*Здесь используются переменные, которые SEO-плагин (Rank Math / Yoast) подтягивает из базы данных (ACF-полей) и заголовка страницы.*

### Для страницы конкретной модели (например, Kia Rio)
* **Переменные:** 
  * `%title%` = Название авто (например, Kia Rio)
  * `%price_from%` = Значение из ACF поля `car_full_price`
* **Title:** Шумоизоляция %title% в Подольске — цена от %price_from% руб., фотоотчет и отзывы | Shumoff
* **Description:** Профессиональная шумоизоляция автомобиля %title% материалами Шумофф. 🛡 Снижение шума до 70%, сохранение дилерской гарантии. Смотрите фото до/после и цены на сайте! Гарантия 12 месяцев.
* **H1 (Заголовок на странице):** Шумоизоляция %title%

## 3. Шаблоны мета-тегов для страниц Услуг (CPT: services)

* **Переменные:**
  * `%title%` = Название услуги (например, Шумоизоляция дверей)
  * `%price_from%` = Значение из ACF поля `service_price_from`
* **Title:** %title% автомобиля в Подольске — цена от %price_from% руб. | Shumoff
* **Description:** Качественная %title% авто в сертифицированном центре. ⏱ Время работ от 1 дня. Официальные материалы, фотоотчет каждого этапа, без потери гарантии на авто. Узнайте стоимость онлайн!
* **H1:** %title% авто

## 4. Рекомендации по контенту (LSI-слова)
Копирайтеру при написании текстов на эти страницы необходимо органично вписывать следующие слова и фразы (LSI):
* *Коммерческие:* цена, стоимость, под ключ, рассчитать, записаться, недорого, прайс.
* *Тематические:* виброизоляция, скрип, пластик, сверчки, акустика, салон, разборка, сборка, материалы, децибелы, антискрип, арки, днище.

## 5. Микроразметка (Schema.org)
* На страницах Услуг и Автомобилей должна быть внедрена разметка `Product` (или `Service` + `Offer`).
* Блок FAQ (Вопрос-ответ) на страницах должен размечаться как `FAQPage`, чтобы занимать больше места в поисковой выдаче Google.
