# Карта сайта Shumoff v2 (WordPress)

Цель: создать сайт, который одновременно работает как:

* коммерческий сайт услуг;
* SEO-платформа;
* каталог выполненных работ;
* генератор заявок.

---

# 1. Структура WordPress

Предлагаю использовать 4 основных типа контента:

### Страницы (Pages)

* Главная
* О компании
* Контакты
* Цены
* Акции
* Политика конфиденциальности

---

### Услуги (Custom Post Type)

`services`

---

### Кейсы (Custom Post Type)

`cases`

---

### Автомобили (Custom Post Type)

`cars`

---

### Статьи (Blog)

Стандартные записи WordPress

# Этап запуска (MVP)

Для запуска достаточно:

### Страницы

* Главная
* Контакты
* Цены

### Услуги

* 6 страниц

### Автомобили

* 20 страниц

### Кейсы

* 20–30 кейсов

### Блог

* 10 статей

Это уже позволит получать заявки и начинать SEO-продвижение. Следующим шагом я бы подготовил **структуру базы данных WordPress (CPT + поля ACF + связи между услугами, автомобилями и кейсами)**, чтобы разработчик мог сразу приступить к сборке сайта без переделок.
