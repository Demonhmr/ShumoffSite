# Shumoff — WordPress Theme

> Профессиональная тема для сайта строительной компании **Shumoff** (Подольск).  
> Ремонт квартир и домов под ключ.

---

## Описание проекта

Shumoff — это кастомная WordPress-тема, разработанная для компании, занимающейся
ремонтно-отделочными работами в Подольске и Московской области. Тема использует
современный стек (CSS Variables, CSS Grid, Flexbox) и полностью адаптирована
под все устройства.

**Ключевые особенности:**

- Адаптивный дизайн (desktop → tablet → mobile)
- Фиксированный header с эффектом при скролле
- Мобильное меню (бургер)
- FAQ-аккордеон
- Калькулятор стоимости ремонта
- Слайдер «До / После»
- Форма обратной связи с валидацией
- Плавная прокрутка к якорям
- CSS-анимации fade-in

## Установка

### 1. Установка плагина shumoff-core

```
1. Скопируйте папку shumoff-core/ в wp-content/plugins/
2. Перейдите в админку WordPress → Плагины
3. Активируйте плагин «Shumoff Core»
```

### 2. Установка темы shumoff-theme

```
1. Скопируйте папку shumoff-theme/ в wp-content/themes/
2. Перейдите в админку WordPress → Внешний вид → Темы
3. Активируйте тему «Shumoff»
```

### 3. Установка Rank Math SEO

```
1. Перейдите в админку → Плагины → Добавить новый
2. Найдите «Rank Math SEO»
3. Нажмите «Установить» затем «Активировать»
4. Запустите мастер настройки SEO
```

### 4. Установка Fluent Forms

```
1. Перейдите в админку → Плагины → Добавить новый
2. Найдите «Fluent Forms»
3. Нажмите «Установить» затем «Активировать»
4. Создайте формы через Fluent Forms → Forms
```

## Настройка

### Настройка меню

```
1. Перейдите в админку → Внешний вид → Меню
2. Создайте новое меню (например, «Основное»)
3. Добавьте страницы в меню
4. В параметрах меню установите «Местоположение» → Header Menu
5. Нажмите «Сохранить меню»
```

### Настройка статической главной

```
1. Создайте новую страницу «Главная»
2. Добавьте блоки/элементы из templates/ в нужном порядке
3. Перейдите в админку → Настройки → Чтение
4. Выберите «Статическую страницу» → Главная: «Главная»
5. Нажмите «Сохранить изменения»
```

### Настройка Google Fonts

Шрифты загружаются автоматически из Google Fonts. Если нужно использовать
локальную загрузку, замените `@import url(...)` в `main.css` на подключение
`.woff2` файлов.

## Кастомизация

### Изменение цветов

Откройте `assets/css/main.css` и измените значения в `:root`:

```css
:root {
  --color-primary: #FF6B00;  /* Ваш новый цвет */
  --color-secondary: #111111;
  /* ... */
}
```

### Изменение ширины контейнера

```css
:root {
  --container-max: 1280px;  /* Ваш новый максимум */
}
```

### Добавление новых страниц

Создайте новую HTML-страницу в `templates/`, скопируйте структуру
из существующего файла и подключите стили/скрипты в `header.php`
и `footer.php` темы.
