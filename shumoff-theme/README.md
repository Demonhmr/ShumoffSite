# Shumoff Theme

Кастомная WordPress-тема сайта **Shumoff Podolsk** — шумоизоляция автомобилей
в Подольске и Москве.

## Возможности

- Адаптивный дизайн (desktop → tablet → mobile), CSS Variables + Grid/Flexbox
- Липкий header, мобильное меню (бургер + оверлей)
- FAQ на нативных `<details>` с микроразметкой FAQPage
- Комплекты Simple / Medium / Premium
- Форма заявки с клиентской валидацией телефона
- SEO: title/description, Open Graph, Twitter Card, JSON-LD
  (AutoRepair, FAQPage, BreadcrumbList)

## Файлы

```
style.css            Заголовок темы + базовые стили и компоненты каркаса
assets/css/main.css  Стили компонентов внутренних страниц (цены, кейсы, поиск)
assets/js/main.js    Скрипты: меню, скролл-эффекты, валидация, слайдер до/после
front-page.php       Главная страница (лендинг)
single-*.php         Детальные страницы: услуга, автомобиль, кейс
archive-*.php        Архивы: услуги, автомобили, кейсы
```

## Палитра и шрифты

| Переменная | Значение | Назначение |
|---|---|---|
| `--color-primary` | `#FF6B00` | Акцент |
| `--color-secondary` | `#111111` | Тёмный фон / заголовки |
| `--color-text` | `#222222` | Текст |
| `--color-bg-alt` | `#F5F5F5` | Альтернативный фон |

Заголовки — Manrope, текст — Inter (Google Fonts).

## Зависимости

Тема работает без плагинов; при активном **ACF Pro** подключаются
дополнительные поля (цены, галереи кейсов, характеристики авто) — доступ к
ним идёт через хелпер `shumoff_field()`, поэтому отключение ACF не ломает
сайт.
